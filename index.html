A couple of things I whipped up.

chüser

Chüser is a decision facilitator web application built with Ruby in Sinatra with integration tests using RSpec and a frontend built with Bootstrap. Chüser plans a day of activities, a night out, or just a quick meal. The application uses the HTML5 geolocation API and basic user inputs to determine parameters for queries. A call is made to the Yelp API using my Yelpify Ruby Gem then serves the user one randomly selected business to accept or reject. A user is able to chain activities, then route to them through a call to the Google Maps API.

Check out the code on GitHub.

Yelpify

Yelpify is a Ruby Gem that simplifies the process of using the Yelp API, and is available on rubygems.org. The gem is complete with integration testing using RSpec for all methods. Yelpify allows users to instantiate new clients, as well as build request queries with easy-to-use method calls. The Yelpify gem works for both search and business requests to the Yelp API. All API responses are returned from Yelp in JSON format that is then parsed into OpenStructs to allow users to procure data using dot-notation. Yelpify has been fully integrated into Chüser.

Check out the code on GitHub.
